Schritt-für-Schritt Setup
  1. Gehe zu console.firebase.google.com → kostenloses Konto erstellen
  2. Neues Projekt erstellen (z. B. "mein-lagersystem")
  3. Im Menü: Realtime Database → Datenbank erstellen → Testmodus wählen
  4. Oben rechts: Zahnrad → Projekteinstellungen → Reiter "Allgemein"
  5. Scrollen zu "Deine Apps" → Web-App hinzufügen (</>) → Config kopieren
  6. Diese HTML-Datei in einem Texteditor öffnen → FIREBASE_CONFIG = null mit deiner Config ersetzen
  7. Datei auf GitHub Pages hochladen → kostenlose URL für alle Firmen
const FIREBASE_CONFIG = {
  apiKey: "AIzaSy...",
  authDomain: "mein-lager.firebaseapp.com",
  databaseURL: "https://mein-lager-default-rtdb.firebaseio.com",
  projectId: "mein-lager"
};
